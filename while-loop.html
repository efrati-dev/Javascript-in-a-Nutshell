Explanation

  • The While Loop

    Loops can execute a block of code as long as a specified condition is true.
    The while loop loops through a block of code as long as a specified condition is true.
    The do while loop is a variant of the while loop.
    This loop will execute the code block once, before checking if the condition is true,
    then it will repeat the loop as long as the condition is true.

  • Comparing For and While

    If you have read the previous chapter,
    about the for loop, you will discover that a while loop is much the same as a for loop,
    with statement 1 and statement 3 omitted. The loop in this example uses a for loop to collect the car names from the cars array.

JavaScript While Loop

Sample code

                
let text = "";
let i = 0;
while (i < 10) {
    text += "
The number is " + i;
    i++;
}
document.getElementById("demo").innerHTML = text;

JavaScript Do While Loop

Sample code

                
let text = ""
let i = 0;

do {
    text += "
The number is " + i;
    i++;
}
while (i < 10);  

document.getElementById("demo").innerHTML = text;
